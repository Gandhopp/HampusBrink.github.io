About Sealoft

Sealoft was my first group project made in Unreal Engine. Here's a short trailer:

My responsibility was to make sure that we had functional NPCs that had unique behaviour. To achieve this goal I made the system for creating NPCs flexible by adding an array of walkpoints that you could place in the world for the NPC to walk to, You can also change how long it stays there and if it is a shop position as seen here:
